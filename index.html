Introducton

This website has been constructed to support the Old Barn Club (OBC) sport competitions. In the past these competitions have been run by members manually using pen/paper and spreadsheets using the Main bar sports notice board. This website will not replace these manual methods but provide electronic support by making data and results available online and possibly in realtime.


The competitions have been run successfully by members of the OBC for many years and this support will be made available ASAP where possible. Some of the compititions will probably remain manual due to the advantages being online not beneficial.


This website is currently under construction and will evolve based on competition timelines, so please bear with us.
